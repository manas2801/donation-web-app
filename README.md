# Donation Web Application

A dynamic donation web app built using Django where users can register, login, and make donations.

## Features
- User login and registration
- Dashboard after login
- Donation page with UPI QR code
- Click QR to view larger
- Donation form
- Thank you page
- Donation history
- Attractive UI

## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript

## Project Structure
donation_project/
manage.py
templates/
static/
app/
requirements.txt

## How to Run
1. Clone the repo:
git clone https://github.com/manas2801/donation-web-app.git

2. Go to folder:
cd donation-web-app

3. Install dependencies:
pip install -r requirements.txt

4. Run server:
python manage.py runserver

5. Open browser:
http://127.0.0.1:8000/
