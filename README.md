# 💰 Donation Web Application

A dynamic web-based donation platform developed using Django. This application allows users to register, login, and make donations easily through a clean and user-friendly interface.

---

## 🚀 Features

- 👤 User Registration and Login system  
- 🏠 Dashboard after successful login  
- 💳 Donation page with UPI ID and QR Code  
- 🔍 Clickable QR Code for enlarged view  
- 📝 Donation form for user input  
- ✅ Thank You page after successful donation  
- 📊 Donation history tracking for users  
- 🎨 Attractive and responsive UI design  

---

## 🛠️ Technologies Used

- Python  
- Django  
- SQLite  
- HTML  
- CSS  
- JavaScript
